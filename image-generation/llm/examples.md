# @radzor/image-generation — Usage Examples

## TypeScript

### Generate with OpenAI DALL-E 3
```typescript
import { ImageGeneration } from "@radzor/image-generation";

const gen = new ImageGeneration({
  provider: "openai",
  apiKey: process.env.OPENAI_API_KEY!,
  model: "dall-e-3",
});

const result = await gen.generate("A cyberpunk city at night, neon lights, rain");
console.log(result.url);           // https://oaidalleapiprodscus...
console.log(result.revisedPrompt); // DALL-E's revised version of your prompt
```

### Generate with Stability AI
```typescript
const gen = new ImageGeneration({
  provider: "stability",
  apiKey: process.env.STABILITY_API_KEY!,
  defaultSize: "1024x1024",
});

const result = await gen.generate("Watercolor painting of a forest");
// Stability always returns base64
const buffer = Buffer.from(result.base64!, "base64");
fs.writeFileSync("forest.png", buffer);
```

### Generate multiple variations
```typescript
const results = await gen.generateMultiple("Logo design for a tech startup", 4);
for (const [i, result] of results.entries()) {
  console.log(`Variation ${i + 1}: ${result.url}`);
}
```

### Event listeners
```typescript
gen.on("onGenerated", ({ url, prompt, provider }) => {
  console.log(`Generated by ${provider}: ${url}`);
  db.insert({ prompt, imageUrl: url, createdAt: new Date() });
});

gen.on("onError", ({ code, message, provider }) => {
  console.error(`[${provider}] ${code}: ${message}`);
});
```

### Get base64 for inline embedding
```typescript
const result = await gen.generate("A small icon of a cat", {
  size: "256x256",
  responseFormat: "base64",
});
const imgTag = `<img src="data:image/png;base64,${result.base64}" />`;
```

## Python

### Generate with OpenAI DALL-E 3
```python
from image_generation import ImageGeneration, ImageGenerationConfig
import os

gen = ImageGeneration(ImageGenerationConfig(
    provider="openai",
    api_key=os.environ["OPENAI_API_KEY"],
    model="dall-e-3",
))

result = gen.generate("A cyberpunk city at night, neon lights, rain")
print(result.url)
print(result.revised_prompt)
```

### Generate with Stability AI
```python
import base64

gen = ImageGeneration(ImageGenerationConfig(
    provider="stability",
    api_key=os.environ["STABILITY_API_KEY"],
    default_size="1024x1024",
))

result = gen.generate("Watercolor painting of a forest")
with open("forest.png", "wb") as f:
    f.write(base64.b64decode(result.base64))
```

### Flask endpoint for image generation
```python
from flask import Flask, request, jsonify

app = Flask(__name__)

gen = ImageGeneration(ImageGenerationConfig(
    provider="openai",
    api_key=os.environ["OPENAI_API_KEY"],
))

@app.route("/api/generate-image", methods=["POST"])
def generate_image():
    data = request.get_json()
    result = gen.generate(data["prompt"])
    return jsonify({"url": result.url, "revised_prompt": result.revised_prompt})
```

### Event listeners
```python
def on_generated(event):
    print(f"Generated by {event['provider']}: {event['url']}")

def on_error(event):
    print(f"[{event['provider']}] {event['code']}: {event['message']}")

gen.on("onGenerated", on_generated)
gen.on("onError", on_error)
```
